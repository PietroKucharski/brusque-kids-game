🎮 Brusque Kids Game

Um jogo educativo interativo sobre a cidade de Brusque, desenvolvido para crianças em ambientes escolares.

📖 Sobre o Projeto

O Brusque Kids Game é uma aplicação web educativa criada para apresentar a história e cultura da cidade de Brusque de forma lúdica e acessível. Por meio de perguntas e alternativas ilustradas, as crianças aprendem sobre a cidade enquanto se divertem.

O projeto foi desenvolvido como parte das atividades acadêmicas do curso de Sistemas de Informação da UNIFEBE (Centro Universitário de Brusque).

✨ Funcionalidades

- ❓ Perguntas interativas sobre Brusque
- 🔊 Botões de áudio para acessibilidade
- 📊 Barra de progresso com mascote (Capivara)
- 🖼️ Imagens ilustrativas por questão
- 🎨 Interface colorida e amigável para crianças

🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Google Fonts (Fredoka, Poppins)
  

🚀 Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/brusque-kids-game.git
   ```

2. Abra a pasta no VS Code

3. Instale a extensão Live Server (Ritwick Dey)

4. Clique com o botão direito em qualquer arquivo `.html` → Open with Live Server
